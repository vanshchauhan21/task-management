📋 Task Management App
A modern, responsive Task Management Application built with React.js. This project allows users to add, edit, delete, and update the status of tasks in a clean and animated interface. It includes beautiful UI transitions, a colorful background, and modal forms with status control.

✨ Features
✅ Add new tasks with name, description, assignee, and status

📝 Edit and update existing tasks

❌ Delete tasks

🔄 Toggle task statuses through:
